## [!DNL Adobe Cloud Platform]{#acp}

Experience Cloud インターフェイスおよび[!UICONTROL プラットフォーム]コアサービスのリリースノートです。Mobile Services、Experience Platform Launch、Dynamic Tag Management、GDPR API および Experience Cloud ID サービスが含まれます。

### [!UICONTROL Dynamic Tag Management]

[!DNL Adobe] では、2020 年末までに [!UICONTROL Dynamic Tag Management] を終了する予定です。

* **2019 年 7 月 1 日：**&#x200B;来年の 7 月から、DTM では新しいプロパティは作成できなくなります。これは、既存のプロパティには影響しません。
* **2020 年 7 月 1 日：** DTM プロパティは読み取り専用モードになります。ツール、ルールおよびデータ要素を作成および編集することはできなくなります。DTM 環境に公開することはできなくなります。これは、以前公開したライブラリには影響しません。
* **2020 年 12 月 31 日：**&#x200B;サポートの終了。サーバーは使用停止となり、ドキュメントはオフラインになって、コミュニティは削除されます。これは、以前公開したライブラリには影響しません。

詳しくは、[DTM Plans for a Sunset](https://forums.adobe.com/community/experience-cloud/platform/launch/blog/2018/10/05/dtm-plans-for-a-sunset) を参照してください。

送信履歴レポートの名前を送信ファイル履歴レポートに更新しました。以前の名前は、一部のお客様に、レポートが HTTP の宛先への送信データを表示するものと誤解させるものでした。実際、レポートは、S3 または FTP の場所に配信されたファイルを扱います。

**既知の問題**

* Safari の最新バージョンには、Intelligent Tracking Prevention（ITP）2.0 ツールが含まれています。これは、Safari ユーザーに対するアドレス可能なオーディエンス指標および h_referer シグナルを使用したデータ収集に影響します。Safari トラフィックについては、[アドレス可能なオーディエンスのマッチ率の低下の原因](https://marketing.adobe.com/resources/help/en_US/aam/addressable-audiences-match-rates.html)および [h_ プレフィックスを使用したデータ収集](https://marketing.adobe.com/resources/help/en_US/aam/r_tb_variable_prefixes.html)を参照してください。
* [アルゴリズムモデリングの特性の除外](https://marketing.adobe.com/resources/help/en_US/aam/trait-exclusion-algo-models.html)のリリースにより、[役割に基づくアクセス制御](https://marketing.adobe.com/resources/help/en_US/aam/c_administration.html)を使用しているお客様に問題が生じます。新しいモデルを作成する際に、アクセス権のあるデータソースのみを選択すると、**除外**&#x200B;ウィンドウに対応する特性を表示できます。ただし、アクセス権のあるデータソースに加えて、アクセス権のないデータソースを選択すると、空のリストが表示されます。特性を表示するには、アクセス権のないデータソースの選択を解除します。（AAM-42380）

**ドキュメントの更新**
